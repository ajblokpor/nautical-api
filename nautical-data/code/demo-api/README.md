# Steps

## Download NodeJS

Download and install NodeJS at [Node JS Website](https://nodejs.org/en).

## Install Express package

Execute `npm install express` in a terminal window.

## Run Demo API

Execute `node ./server.js` in a terminal window.

To include berth positions in the API responses, execute `npm run startWithPositions` or `node ./server.js --with-positions`.

To inspect the available command line options, execute `npm run help` or `node ./server.js --help`.
